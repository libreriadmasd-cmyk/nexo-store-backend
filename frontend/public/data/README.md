# Carpeta de datos — Nexo Store

Subí acá el archivo `productos.json` con el siguiente formato exacto:

```json
[
  {
    "id": 1,
    "nombre": "Nombre del producto",
    "precio": 1234.56,
    "sku": "7794765001770",
    "stock": 10,
    "imagen": "7794765001770.jpg",
    "categoria": "Escolar"
  }
]
```

- El campo `imagen` debe coincidir exactamente con el nombre del archivo en `/public/images/`.
- Las categorías sugeridas son: `Escolar`, `Oficina`, `Juguetería`, `Marroquinería`.
