# Carpeta de imágenes — Nexo Store

Subí acá las imágenes de los productos. El nombre de cada archivo debe coincidir EXACTAMENTE con el valor del campo `imagen` en `productos.json`.

Ejemplo: si el producto tiene `"imagen": "7794765001770.jpg"`, entonces el archivo debe llamarse `7794765001770.jpg` y estar dentro de esta carpeta.

Formatos aceptados: `.jpg`, `.jpeg`, `.png`, `.webp`.
